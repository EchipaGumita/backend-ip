API Documentation

Below are the API routes for the project along with example requests. You can copy and paste them as needed.

Get All Exam Requests

Method: GET

URL: /exam-requests

Description: Fetch all exam requests with their details.

Example:

curl -X GET http://localhost:5000/exam-requests

Create Exam Request

Method: POST

URL: /exam-requests

Description: Create a new exam request.

Example:

curl -X POST http://localhost:5000/exam-requests \
-H "Content-Type: application/json" \
-d '{
    "studentUniqueId": "12345",
    "subject": "Math",
    "examDate": "2024-12-01",
    "examDuration": 90,
    "classroom": "64a0f87595d09c5e648eeb32",
    "hour": "10:00",
    "mainProfessor": "64a0f87595d09c5e648eeb33",
    "secondaryProfessor": "64a0f87595d09c5e648eeb34",
    "faculty": "Computer Science",
    "group": "64a0f87595d09c5e648eeb35"
}'
